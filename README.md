#  ISTI-Web — Custom Website Framework

> Fast, accessible, SEO-optimized website template powering ISTI client sites.

## Overview

Production-ready website scaffold used across ISTI client projects. Built for performance, accessibility, and rapid customization per client brand.

## Stack

- **Frontend:** React 19 · TypeScript · Tailwind CSS v4 · Vite
- **Routing:** React Router v6
- **SEO:** React 19 native head hoisting (no Helmet needed)
- **Styling:** Tailwind CSS with custom design token system
- **Deployment:** Vercel / Netlify / Railway

## Features

- ⚡ Sub-second load times (Vite build + code splitting)
- 📱 Mobile-first responsive layout
- 🔍 SEO-ready (meta, OG, Twitter card, structured data)
- ♿ WCAG 2.1 AA accessibility
- 🎨 Brand token system (colors, typography, spacing)
- 📊 Google Analytics 4 integration ready

## Project structure

```
src/
  client/
    components/     # Reusable UI components
    pages/          # Route-level page components
    lib/            # Utilities (cn, hooks, helpers)
    assets/         # Brand assets (logo, favicon)
  server/           # API layer (Node.js)
```

## Quick start

```bash
npm install
npm run dev
```
